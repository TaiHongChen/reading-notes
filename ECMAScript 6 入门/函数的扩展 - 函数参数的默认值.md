# 函数的扩展 - 函数参数的默认值

## 基本用法

```javascript
function sum(x, y = 10) {
  return x + y;
}

console.log(sum(1)); // 11
console.log(sum(1, 2)); // 3

function Person(name = 'hanzichi', age = 30) {
  this.name = name;
  this.age = age;
}

let p = new Person();
console.log(p);
// Person { name: 'hanzichi', age: 30 }
```
